Java Notes

What is Java?

Java is an object-oriented programming language used to develop software applications.

Why is Java called an OOP Language?

Because it follows the core principles of Object-Oriented Programming:

  • Abstraction
  • Polymorphism
  • Inheritance
  • Encapsulation

Language Basics

Language: A medium used to communicate between two entities.

Programming Language: A medium used for communication between humans and machines.

Tokens in Java

Tokens are the smallest units of a Java program.

  • Names given to classes, variables, and methods
  • Cannot be Java keywords
