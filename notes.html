Java Notes

What is Java?
Java is an object-oriented programming language used to develop software applications.

Why is Java called OOPL?
Because it follows the principles of Object-Oriented Programming (OOP), such as:
  1. Abstraction
  2. Polymorphism
  3. Inheritance
  4. Encapsulation
What is a language?
A language is a medium used to communicate between two entities.

What is a programming language?
A programming language is a medium that enables communication between humans and machines.

How is a programming language built? 
[Data Types] → [Tokens] → [Statements] → [Programs] → [Software]

Tokens

Tokens are the smallest units of a Java program.
  1. Identifiers
    • An identifier is the name given to a class, variable, or method.
    • Identifiers cannot be Java keywords.
  2. Keywords
    • Keywords are predefined words that have special meaning in Java.
    • Java has 50 reserved keywords.
  3. Literals
    • Literals are fixed values used in a program.
    • Types of literals:
    1. Numeric Literals
      1. Integer literals → 1, 2, 100
      2. Decimal literals → 1.1, 0.9
    2. String literals → "Hello", "9"
    3. Character literals → 'A', '9'
    4. Boolean literals → true, false
  4. Operators

    Operators are symbols used to perform operations.

    1. Unary Operators (Prefix, Postfix)
    2. Arithmetic Operators (Additive, Multiplicative)
    3. Shift Operators
    4. Relational Operators (Comparison, Equality)
    5. Bitwise Operators (AND, OR, XOR)
    6. Logical Operators (&&, ||)
    7. Ternary Operator
    8. Assignment Operators
  5. Separators
    • { } → Braces
    • [ ] → Brackets
    • ( ) → Parentheses
    • ; → Semicolon
    • , → Comma
  6. Comments

    Comments provide additional information in a program.

    1. Single-line comment 
      // This is a single line comment
    2. Block comment 
      /* Multi-line
   Comment */

Basic Logic Examples

1. int a = 10;
   print a;
   Output: 10

2. int b = 20;
   print b;
   Output: 20

3. int a = 10;
   int b = 20;
   print a + b;
   Output: 30

4. int x = 20;
   int y = 40;
   int z = x - y;
   print z;
   Output: -20

5. int a = 10;
   int b = a;
   print a + b;
   Output: 20

6. int b = 8;
   int a = b;
   int z = a;
   print a + b + z;
   Output: 24

7. int a = 10;
   int b = a - 5;
   print a + b;
   Output: 15

Java Types

Java has four main types:

  1. Class
  2. Interface
  3. Enum
  4. Annotation

Class

A class is a blueprint used to create objects.
It must be saved with the .java extension.

Compilation

Syntax: 
javac FileName.java

Execution

Syntax: 
java FileName

Main Method Variations

  1. public static void main(String[] args)
  2. static public void main(String[] args)
  3. public static void main(String args[])
  4. public static void main(String... args)

Scope

Variables declared inside a block are not accessible outside the block. 
{
 int a = 10;
 System.out.println(a);
}
Output: 10

Class Members

  1. Variables (Data Members) → store data
  2. Methods → perform operations
  3. Constructors → initialize variables

Variables

A variable is a named memory location used to store data.

Types of Variables

Primitive Types
  • byte
  • short
  • int
  • long
  • float
  • double
  • char
  • boolean
Non-Primitive Types
  • Array
  • String

Variable Operations

1. Declaration 
int a;
2. Initialization 
a = 10;
3. Declaration + Initialization 
int a = 90;
4. Reinitialization 
int a = 10;
a = 90;
System.out.println(a);
Output: 90
5. Copying Variables 
int a = 90;
int b = a;
System.out.println(b);
Output: 90

Local variables

    1. any variable which is declared within the method is called as Local variables . 
    2. the scope of the local variables is form begining of the method till end of the method. 
    3. it will not have default values . 
    4. it cannot be classified into static and non-static . 
    5. local variables should be intilised before utilisation 

    example :
    class Sample
    {
        public static void main(String [] args)

            int a = 10;


            System.out.println(a);
        }

    }

Global variables 

    1. any variables declared outside the method and insdie the class is called Global variables. 
    2. the scope of the global variables is start of the class till end of the class . 
    3. it can be classified into static and non static 
    4. once the global variable declared imidietly in the next line we cannot initilize or reinitlize .  
    5. it will have default values .
